# tvOS

Transluscent GTK theme based on the appearance of Apple tvOS

![tvos-13](https://b00merang.weebly.com/uploads/1/6/8/1/16813022/screenshot-from-2018-05-15-16-34-55_orig.png)

**More info :** http://b00merang.weebly.com/tvos.html

### Manual installation

Extract the zip file to the themes directory i.e. `/home/USERNAME/.themes`

### Requirements

- GTK+ 3.20 or above
- Murrine and Pixmap theme engines
